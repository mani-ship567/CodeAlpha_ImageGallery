# CodeAlpha Image Gallery

## Project Description
This is a responsive Image Gallery created using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features
- Responsive Image Gallery Layout
- Image Categories (Nature & Animals)
- Filter Buttons
- Lightbox View
- Next and Previous Navigation
- Hover Effects
- Smooth Transitions

## Technologies Used
- HTML5
- CSS3
- JavaScript
